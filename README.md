# Ruby Cowsay
[![License](http://img.shields.io/:license-gpl3-blue.svg)](http://www.gnu.org/licenses/gpl-3.0.html)

Cowsay for Ruby

## How to use

Run cowsay.rb from your command line. No need to install any gems!

```
$ ./cowsay.rb "Moo, I am a cow"

  ---------------
< Moo, I am a cow >
  ---------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
```
